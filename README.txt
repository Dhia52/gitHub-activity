Ceci est une activité d'initiation à l'utilisation de GitHub ainsi qu'au versionnage du code.

Le but de l'exercice est de créer un répository en ligne :

	ce répository contient un fichier README.txt qui contient l'énoncé de l'activité.
	il contiendra aussi deux autres fichiers au choix de l'étudiant.
	il doit avoir plus de 4 commits distincts et la description des commits doit bien indiquer les changements qui ont eu lieu.
